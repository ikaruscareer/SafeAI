# MCP Examples

Examples demonstrating Model Context Protocol configurations.

## What these demonstrate:
- MCP server definitions (v1.0 and v1.1 schemas)
- MCP client configurations
- Tool and resource definitions
- Authentication configurations (missing, weak, token-based)
- Transport configurations (stdio, HTTP, WebSocket)
- Exposed endpoints
- Hardcoded secrets in MCP configs

Add `.json` and `.yaml` files showing MCP configuration patterns.
