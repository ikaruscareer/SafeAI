# Risky Examples

Examples that demonstrate **risky** AI agent patterns.

## What these demonstrate:
- Direct user input interpolation into prompts
- Shell execution with uncontrolled parameters
- Hardcoded API keys, tokens, and passwords
- Missing authentication on MCP endpoints
- Autonomous loops without iteration limits
- Unrestricted filesystem and database access
- Missing timeout and retry configurations

Add `.py`, `.json`, or `.yaml` files showing risky agent configurations.
